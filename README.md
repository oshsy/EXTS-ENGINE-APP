**EXTS ENGINE** is a utility for both non-rooted and rooted Android devices, designed for manual and automatic hardware resource management. It uses a **Logic Bot** to dynamically adjust performance profiles, display resolution, and system limitations based on user activity.

---

## Features

### Performance Modes

*   **1. Power Saving Mode**: Prioritizes battery life, lower temperatures, and quieter system operation by reducing performance.
*   **2. Daily Mode**: Balanced settings for comfortable everyday use.
*   **3. Medium Mode**: Maintains stable and balanced performance.
*   **4. Extreme Mode**: Pushes the device beyond its normal limits for demanding games and heavy workloads.

### Smart System Management

*   **5. Automatic Screen Resolution Adjustment**: Dynamically adjusts display resolution (especially in Extreme Mode) to reduce CPU and GPU workload.
*   **6. Automatic System Management**: The Logic Bot analyzes hardware conditions and automatically boosts performance when whitelisted applications are running.
*   **7. Background Process Limiter**: Restricts non-whitelisted applications to free up system resources.
*   **8. Blacklist App Blocking**: Prevents blacklisted applications from running and consuming resources.
*   **9. Sensor Shutdown Mode**: Disables the camera, microphone, gyroscope, compass, and other sensors for enhanced privacy.

### Display & Graphics Controls

*   **10. Manual Resolution Control**: Allows users to manually adjust screen resolution, from native quality down to ultra-low graphics settings for higher performance.
*   **11. Force Refresh Rate Control**: Lets users manually set the display refresh rate according to hardware capabilities.
*   **12. Manual Graphics Driver Selection**: Enables custom graphics driver configurations for whitelisted applications to maximize performance.
*   **13. Synthetic Compositor**: Improves frame smoothness and visual consistency but requires additional RAM.
    > **Note**: This feature does **not increase game FPS**. It only smooths existing frames. If a game is already experiencing severe frame drops, this feature will provide little or no benefit.
*   **14. Manual Display Resolution Adjustment**: Lowers screen resolution to improve performance at the cost of visual quality.

---

## Tweaks & Hardware Controls (Root Required)

*   **15. Thermal Limit Removal**: Removes thermal throttling restrictions to maintain maximum performance.
*   **16. High Performance CPU Mode**: Forces CPU frequencies to remain at their highest available levels.
*   **17. High Performance GPU Mode**: Forces GPU frequencies to remain at their highest available levels.

## Requirements

### Non-Root Features
*   Android 11.0+
*   Shizuku support (recommended)

### Root Features
*   Magisk / KernelSU / APatch
*   Root access granted
