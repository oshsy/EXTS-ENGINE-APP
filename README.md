# EXTS-ENGINE

**EXTS ENGINE APP** adalah utilitas berbasis no root dan root untuk manajemen resource hardware secara manual dan otomatis. Menggunakan *Logic Bot* untuk mengatur profil performa, resolusi, dan limitasi sistem berdasarkan aktivitas pengguna.

---

1. **Mode Hemat Daya:** Fokus pada penghematan baterai, suhu dingin, dan sistem yang tenang (Performa diturunkan).
2. **Mode Harian:** Keseimbangan untuk kenyamanan penggunaan aplikasi sehari-hari.
3. **Mode Medium:** Menjaga performa tetap stabil dan seimbang.
4. **Mode Extreme:** Memaksimalkan perangkat sampai titik di luar batas untuk gaming berat.
5. **Otomatis Ubah Resolusi Layar:** Menyesuaikan resolusi (khususnya di Mode Extreme) untuk meringankan beban CPU/GPU.
6. **Otomatis Kelola System:** Logic bot melakukan kalkulasi berdasarkan kondisi hardware. Jika aplikasi di-*whitelist* jalan, performa langsung ditingkatkan otomatis.
7. **Batasi Latar Belakang:** Membatasi aplikasi non-whitelist hingga blacklist agar resource tetap lega.
8. **Blokir Aplikasi Blacklist:** Memaksa aplikasi yang di-blacklist tetap diam dan tidak berjalan.
9. **Nonaktifkan Sistem Sensor:** Mematikan kamera, mic, giroskop, kompas, untuk privasi total.

10. **Atur Resolusi Manual:** Mengedit resolusi layar dari normal hingga grafik buram/low demi FPS tinggi.
11. **Paksa Refresh Rate Manual:** Mengatur Hz layar sesuai keinginan dan kemampuan hardware.
12. **Pilih Driver Grafis Manual:** Mengatur driver grafis khusus untuk aplikasi di whitelist guna performa maksimal.


### Tweaks & Hardware (Root Required)

13. **Tanpa Batasan Thermal:** Melepaskan batas suhu hardware untuk performa tanpa throttling.
14. **Mode CPU Performa Tinggi:** Memaksa frekuensi CPU ke titik tertinggi tanpa turun.
15. **Mode GPU Performa Tinggi:** Memaksa frekuensi GPU ke titik tertinggi tanpa turun.

---

### ⚠️ Disclaimer
**Developer** tidak bertanggung jawab atas kerusakan hardware, overheat, atau bootloop. Penggunaan beberapa fitur hingga fitur **Tanpa Batasan Thermal** dan **Mode Extreme** berisiko tinggi. **Use at your own risk!**
